# 前端开发指南

## 初始化项目
## 1\. 安装依赖，先进入项目目录，输入以下指令

> npm install

## 2\. 编译

> npm run build

## 3\. 本地开发

> npm run dev
> 本地打开 http://localhost:8118


# 项目结构
```
├── src
│   ├── assets                    # 图片资源
│   ├── styles                    # 样式表
│   ├── views                     # 所有路由组件
│   ├── ...
│   ├── App.vue                   # 根组件
│   ├── resources.js              # 接口配置文件
│   ├── router.js                 # 路由配置文件
│   └── main.js                   # 主入口
│
├── dist                          # 打包编译后的文件
│
├── node_modules                  # 依赖包
│
├── test                          # 测试用例
│
└── build
    ├── ...
    ├── webpack.base.config.js    # webpack基础配置
    ├── webpack.dev.config.js     # webpack开发环境配置
    └── webpack.prod.config.js    # webpack生产环境配置
```
